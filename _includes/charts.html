Meus conhecimentos

JavaScript everywhere!

Criado originalmente para rodar apenas em navegadores, o JavaScript superou suas amarras e já é considerado a linguagem do futuro por muitos especialistas. Hoje a linguagem também passeia por servidores (NodeJS), celulares (PhoneGap), aplicativos desktop (Electron), serviços de Data-mining (CasperJS), Banco de dados (MongoDB) e muito mais.
